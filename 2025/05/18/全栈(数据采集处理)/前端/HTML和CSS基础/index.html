CSS Flexbox 和 Grid 布局详解

Flexbox 和 Grid 是现代 CSS 中两种强大的布局系统，它们彻底改变了我们创建网页布局的方式。下面我将详细介绍这两种布局模型。

1. Flexbox (弹性盒子布局)

Flexbox 是一种一维布局模型，适合在单个行或列上排列元素。

基本概念

主轴(main axis) 和 交叉轴(cross axis) 是 Flexbox 的两个核心概念：

  • 主轴由 flex-direction 定义
  • 交叉轴垂直于主轴

容器属性

  1. flex-direction: 定义主轴方向

  2. flex-wrap: 控制是否换行

  3. justify-content: 主轴对齐

  4. align-items: 交叉轴对齐

  5. align-content: 多行内容对齐

项目属性

  1. order: 控制项目顺序

  2. flex-grow: 定义放大比例

  3. flex-shrink: 定义缩小比例

  4. flex-basis: 定义初始大小

  5. flex: 简写属性

  6. align-self: 单个项目对齐

使用场景

  • 导航栏
  • 等高列布局
  • 垂直居中
  • 响应式布局中的元素重排

2. CSS Grid (网格布局)

Grid 是二维布局系统，可以同时处理行和列。

基本概念

Grid 布局由 网格容器、网格项目、网格线、网格轨道 和 网格单元格 组成。

容器属性

  1. grid-template-columns/grid-template-rows: 定义列和行

  2. grid-template-areas: 定义区域模板

  3. grid-gap (grid-row-gap/grid-column-gap): 网格间距

  4. justify-items: 行轴对齐

  5. align-items: 列轴对齐

  6. justify-content/align-content: 网格对齐

  7. grid-auto-columns/grid-auto-rows: 隐式轨道大小

  8. grid-auto-flow: 自动放置算法

项目属性

  1. grid-column/grid-row: 项目位置

  2. grid-area: 项目区域

  3. justify-self/align-self: 单个项目对齐

特殊单位

  1. fr 单位: 分数单位

  2. minmax() 函数: 定义大小范围

  3. repeat() 函数: 重复模式

使用场景

  • 复杂网页布局
  • 杂志式排版
  • 仪表盘
  • 任何需要精确二维控制的设计

Flexbox vs Grid 对比

特性 Flexbox Grid
维度 一维 二维
方向 单行或单列 行和列同时控制
对齐 基于主轴和交叉轴 基于行和列
项目大小 基于内容 可以显式定义
适合场景 小规模布局，组件内部 大规模布局，整体页面结构
嵌套需求 常需要嵌套实现复杂布局 单一网格容器可处理复杂布局
浏览器支持 广泛支持 现代浏览器支持

实际应用建议

  1. 结合使用：通常 Grid 用于整体页面布局，Flexbox 用于组件内部布局
  2. 渐进增强：对旧浏览器提供回退方案
  3. 命名网格线：提高代码可读性
  4. 响应式设计：使用媒体查询调整网格或弹性布局
