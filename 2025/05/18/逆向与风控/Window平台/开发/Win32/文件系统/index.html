文件系统

卷 (Volume)

概念

卷是 Windows 存储体系中的最高层级单位，指代一个独立的存储设备或分区：

  • 可以是物理硬盘分区 (如 C:、D:)
  • 也可以是虚拟存储设备 (如网络映射驱动器)
  • 每个卷有唯一的卷名和卷标

基本操作

目录 (Directory)

概念

目录是文件系统中用于组织文件的容器：

  • 采用树形结构
  • 支持嵌套子目录
  • 路径分隔符为反斜杠 ()

基本操作

文件 (File)

概念

文件是存储数据的基本单位：

  • 由文件名唯一标识
  • 具有属性和安全描述符
  • 支持多种访问模式

基本操作

内存映射文件 (Memory-Mapped File)

概念

内存映射文件是将磁盘文件映射到进程地址空间的技术：

  • 允许像访问内存一样访问文件
  • 提高大文件读写效率
  • 支持进程间共享数据

基本使用

内存映射文件优势

  1. 高效IO：避免频繁的read/write系统调用
  2. 随机访问：像操作内存一样随机访问文件任意位置
  3. 数据共享：多个进程可映射同一文件实现通信
  4. 延迟加载：实际按需加载物理页，节省内存

应用场景

  • 处理超大文件(如视频编辑)
  • 进程间共享数据
  • 实现高效数据库存储
  • 加载DLL/EXE等可执行文件
