案例二：伪元素

练习网址：CSS伪元素反爬_N04_Spiderbuf

CSS伪元素详解：

CSS 伪元素 | 菜鸟教程

CSS 中 ::before 和 ::after 伪元素的 7 种实用方法 | CSS-Tricks - CSS技巧

CSS 伪元素反爬虫（CSS pseudo-element anti-scraping）技术是通过使用 CSS 伪元素（如 ::before 和 ::after）来隐藏或动态生成内容，从而使得爬虫无法直接抓取页面的真实内容。这种技术利用了伪元素的特殊性质，通过在页面上动态插入内容、装饰或修改文本显示，干扰爬虫的抓取。

原理：

伪元素 ::before 和 ::after 是在页面元素的内容前后插入额外的内容，它们对页面的实际 DOM 结构没有影响，但可以在视觉上进行显示或改变。通过 CSS 来修改内容，爬虫在抓取页面时只会获得经过样式渲染后的文本内容，而难以看到通过伪元素动态插入的真实数据。

具体应用：

  1. 动态生成内容： 使用伪元素来插入文本或链接，伪装页面内容。爬虫抓取到的页面内容并不包含这些伪元素，所以无法获取真实信息。
  2. 内容分割与隐藏： 将真实的内容通过伪元素显示给用户，但实际的 DOM 结构中并不包含这些内容，这样爬虫就无法直接抓取页面中的信息。

生活中的例子：

想象你有一本书，你把书的文字内容分成了两部分，第一部分是普通的文本，第二部分是通过一层透明的薄纸遮盖住的。正常情况下，人眼可以看到透明纸上的文字，但如果有人试图扫描这本书（相当于爬虫），他只能看到普通部分的内容，而无法扫描到透明纸下的文字内容。

在网页中，类似的技术通过伪元素将页面的真实内容隐藏或以另一种方式呈现，阻止爬虫直接抓取页面的所有信息。
